1. Install dulu virtual environment
	a. CD ke folder 'web_flask' di local disk C
	b. buka cmd
	c. jalanin perintah py -m venv <nama folder virtual environment>
	   nama folder virtual environmentnya = venv
	   jadi perintahnya : py -m venv venv
	d. ketik perintah venv\Scripts\activate untuk menjalankan virtual environment


2. Install flask
	a. menjalankan perintah pip install flask

3. Setting file python yang mau menjadi program utama flask 
	a. buat file python namanya app.py utk jadi program utama flask
	b. jalankan perintah set FLASK_APP=app.py

4. Setting environment menjadi development
	a. jalanin perintah set FLASK_ENV=development

5. Jalanin server flask utk bisa buka website yang kita buat
	a. jalanin perintah flask run --host 0.0.0.0

6. Library lain:
	- pip install flask-restful // buat REST api
	- pip install flask-mysqldb // buat database
